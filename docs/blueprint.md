# **App Name**: Afsheen's Enchanted B-Day

## Core Features:

- Personalized Birthday Message Generation: Generates a custom, heartfelt birthday message based on the age (17) and name (Afsheen), and incorporates a personalized touch about the sender's crush.
- Interactive Timeline: Displays key life events leading up to Afsheen's 17th birthday using her date of birth (10 Nov 2008) and incorporating elements based on info that user provided.
- Digital Confetti: Visually engaging animation of colorful confetti that responds to user interactions (e.g., mouse movements or taps).
- Dynamic Age Display: Calculate and display Afsheen's current age in years, months and days using her date of birth, as well as an interactive countdown of the number of days till her next birthday
- Heartfelt AI Tool: Provides thoughtful sentiments and kind affirmations based on an understanding of a teen girl like Afsheen.  The tool adapts its tone and content to reflect what makes this milestone important to Afsheen.

## Style Guidelines:

- Primary color: Lavender (#E6E6FA) for a soft and dreamy feel.
- Background color: Pale Lavender (#F5F5FF), a lighter tint of lavender for a clean background.
- Accent color: Dusty Rose (#D8BFD8), adding warmth and charm.
- Body and headline font: 'Belleza' for a fashionable, artsy, personal touch; 'Alegreya' will be used in the case of larger blocks of text.
- Use soft, pastel-colored icons (hearts, stars, cakes) throughout the website.
- A clean, modern layout with a focus on readability and visual appeal.
- Smooth, subtle animations to enhance user experience (e.g., fade-in effects, gentle transitions).
